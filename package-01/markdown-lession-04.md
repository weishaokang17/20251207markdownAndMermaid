# 删除线


**语法**：

**两个波浪线，后面是被删除文字，再往后，又是两个波浪线。即被波浪线包裹的部分，就是被删除的文字**



~~被删除文字~~


 它~~明确~~地向读者（和机器）传达一个信息：这段内容不是作者原创的叙述或观点，而是引述自其他来源、上下文、人物或作为需要特别强调/隔离的内容

# 下划线

**语法：下划线可以通过 HTML 的 <u> 标签来实现：**

<u>带下划线文本</u>


# 文本高亮
文本高亮不是标准 Markdown 语法，但许多扩展支持：

常见语法（部分平台支持）：

这是==高亮文本==

HTML 替代方案：

这是<mark>高亮文本</mark>

mark标签是一定能生效的，上面的一对==包裹的，只有部分平台支持
