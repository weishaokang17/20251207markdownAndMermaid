# 段落与换行、空行的作用

## 一 ：中间没有换行

这是第一行
这是第二行

上面可以看出来，编辑时的正常的换行， 渲染效果，是不会换行的

## 二 ： 段落
>**语法：中间隔一个空行，同一段文字，就变成了两个段落**

这是第一行

这是第二行

这是第三行

**从中看出来，中间没有空行，就不会换行，有了空行
就会换行，但有一行空行、和有多行空行，效果是一样的。  
如果要多空几行，需要借助 HTML 的 br标签**


## 三 : 段内换行
>**语法一：当前这一行的后面，空两个空格，然后另起一行，开始写其他文字。两个空格会把两段文字段内换行**  

我是第一行，如果想要段内换行，需要该行最后至少两个空格，然后另起一行  
我是第二行

**从中看出来，如果是想要换行，但换行后，显示的效果是中间没有空行，也就是说是段内换行。就在第一行的最后至少有两个空格，然后另起一行，这才能实现段内换行**

**语法二：使用HTML 换行标签<br>**

例如：<br/>
第一行内容<br>
第二行内容

**语法三：反斜杠（部分解析器支持）**

第一行内容\
第二行内容


# 四、空行的作用

空行在 Markdown 中扮演重要角色,有两种作用，分隔段落和隔离不同元素

1.分隔段落：

第一段内容

第二段内容

2.分隔不同元素：

### 标题

段落内容

- 列表项1
- 列表项2

另一段内容
