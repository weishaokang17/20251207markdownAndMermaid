# 警告

**markdown 中GFM 语法，五种标准警告信息类型，详细介绍如下：
Note、Tip、Warning、Important、Caution**

> [!TYPE]
> 正文内容

其中 TYPE 的取值，是上面五种

| 类型            | 强调程度  | 典型场景           |
| ------------- | ----- | -------------- |
| **NOTE**      | ★☆☆☆☆ | 补充说明、不重要但有用的信息 |
| **TIP**       | ★☆☆☆☆ | 建议、技巧、最佳实践     |
| **IMPORTANT** | ★★★☆☆ | 必须遵守的重要条件      |
| **WARNING**   | ★★★★☆ | 有风险，可能导致错误或损坏  |
| **CAUTION**   | ★★★★★ | 高危险、不可逆操作、严重后果 |

>[!NOTE]

>[!TIP]

>[!IMPORTANT]

>[!WARNING]

>[!CAUTION]
