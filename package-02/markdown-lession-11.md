# 跨文档锚点演示

**语法:**

```
[链接文字](路径/文件名.md#锚点ID)
```

例如：

[lession-06中的其中一个标题](../package-01/markdown-lession-06.md#在线教程)

> 注意：跨文档的锚点，只支持markdown 的标题的锚点，不支持跨文件的HTML锚点
